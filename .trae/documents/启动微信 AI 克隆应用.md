# 启动微信 AI 克隆应用

## 项目介绍
这是一个基于 React + TypeScript + Vite 开发的微信 AI 克隆应用，集成了 Google Gemini API 提供 AI 功能。项目包含聊天窗口、群聊创建器、朋友圈、新朋友、侧边栏和贴纸管理器等组件，以及数据库服务、Gemini 服务和 P2P 服务。

## 启动计划
1. 安装项目依赖
2. 检查 GEMINI_API_KEY 配置
3. 启动开发服务器
4. 验证应用是否正常运行

## 具体步骤
1. 运行 `npm install` 安装所有依赖
2. 检查 `.env.local` 文件中的 GEMINI_API_KEY 是否已配置（已存在且配置）
3. 运行 `npm run dev` 启动开发服务器
4. 访问输出的本地 URL 验证应用是否正常运行
